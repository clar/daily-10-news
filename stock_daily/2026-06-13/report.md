## 重点点评

### 🔑 1. SpaceX IPO 首日 +19% — "1.75 万亿登场"对一级 / 二级市场都是新坐标

**[CNBC: SPCX IPO 实时报道](https://www.cnbc.com/2026/06/12/spacex-ipo-spcx-live-updates.html)** · **[Yahoo Finance: SpaceX 首日交易](https://finance.yahoo.com/markets/stocks/articles/spacex-set-begin-trading-raising-132927221.html)**

SpaceX 以 $135 / 5.556 亿股 Class A 定价完成 IPO，募资 $75B，市值 $1.75 万亿——直接成为美国市值第 7 大公司，超过 Tesla。首日开盘 $150（+11%），盘中冲到 $176.52（+30.8%），收报 $160.95（+19%）。今日 MSCI 指数将 SPCX 快速纳入；但 S&P Dow Jones 拒绝豁免，要求 4 个连续盈利季度（SpaceX 2025 净亏 $4.94B），意味着至少要等到 2027 年中才能进 S&P 500——这阻断了 ~$27B 的被动买入。

更重要的是这次 IPO 把"美股流动性消化能力"测试了一遍。SpaceX 同期已经叠加 OpenAI ($852B 估值) 与 Anthropic ($965B 估值) 即将上市的预期，三家合计 ~$3.5T。Goldman Sachs 的测算是 $8T 的货币市场基金能完全吸收，但 Magnificent 7 本月已经蒸发 $2T 市值，被动基金腾仓位的过程实际比预想更剧烈。

下一步看点：1）SPCX lockup 期 366 天对早期股东构成强约束，流通盘只有 4%，短期价格"挤压" vs. 7/7 加入 Nasdaq-100 的买入需求会形成对冲；2）OpenAI / Anthropic IPO 时点是否会因 SpaceX 已"吃掉"散户筹码而推迟到 2027；3）S&P 500 2027 年 SPCX 入选的硬约束，将让 SPCX 与 Tesla 的市值博弈成为长期叙事。

**角度：** SpaceX 不只是上市一个公司，是把"AI 基建 + 太空 + 军工"打包成全美第 7 大公司，整个二级市场的"风险偏好坐标系"被重置了一档。

### 🔑 2. Oracle capex 飙至 $55.7B — "AI 重资产化"反噬，市场开始质疑回报周期

**[The Street: ORCL 财报反应](https://www.thestreet.com/latest-news/orcl-oracle-earnings-call-updates-q4-2026)** · **[CNBC: Oracle Q4 2026](https://www.cnbc.com/2026/06/10/oracle-orcl-q4-earnings-report-2026.html)**

Oracle Q4 营收 $19.18B、EPS $2.11，双双小幅超市场预期。但盘前 -9.4% 至 $184.43，原因不在收入——而在两个数字：1）FY2026 全年资本开支 **$55.7B**，同比 +162%，远超原定 $50B 计划；2）公司宣布将通过债务 + 股票融资 $40B（含已宣布的 $20B 股票发行）。同期未平履约金额 (RPO) **$638B**，YoY +363%——其中绝大部分是 AI 基础设施大单，客户预付 GPU 或直接自带 GPU。

数字背后的市场反问是：以 Oracle 当前的 EBITDA 体量 ($30B+) 撑得起一年 $55B 的 capex 吗？$67B 新 AI 合同里 BYO-GPU 模式占大头，意味着 Oracle 收的是托管费、不是芯片差价毛利。下一财年要让 ~1GW 新增算力上线——按当前 H100/B100 服务器折旧周期 5–6 年算，每 GW 折旧约 $5–6B/年，对 Oracle 自由现金流的拖累比预期更重。

更深的信号是市场情绪转变：去年这种 capex 数字会被解读成"AI 需求强劲"，今年同样数字被解读成"赌注过重 / 摊薄风险加大"。这是 AI 基建股第一次出现明显的"capex 翻倍 = 股价跌"反应，下一个验证就是 7 月 Microsoft 与 Amazon 财报里的 AWS / Azure capex 指引——如果三家头部 hyperscaler 一起把 capex 上调 50%，整个 AI 基础设施板块的估值倍数会进一步压缩。

**角度：** AI 这一波 capex 周期已经走到了"需要证明回款"的阶段——Oracle 是第一家因 capex 暴涨被市场惩罚的大盘股，但绝不会是最后一家。

### 🔑 3. Adobe Q2 双超预期却跌 6.8% — "利润率叙事"取代"AI 叙事"成为软件股新评分卡

**[GuruFocus: ADBE Q2 财报](https://www.gurufocus.com/news/8910656/adobe-adbe-earnings-report-set-for-thursday-amid-market-tensions)** · **[Adobe 8-K SEC 文件](https://www.sec.gov/Archives/edgar/data/0000796343/000079634326000109/adbeex991q226.htm)**

Adobe Q2 营收 $6.62B、调整后 EPS $5.96，均超 Street 预期。但 non-GAAP 营业利润率 44%（低于预期 44.5%），加上 CFO Dan Dunn 6/15 离任另谋发展——盘后股价跌 6.8%。表面看是 0.5 个百分点的利润率差距，实质上反映了软件股市场新的评分标准：**AI 投入带来的 GenAI 收入增量必须明显追平 AI 训练 / 推理增量成本，否则"AI 叙事"立刻让位给"margin compression"叙事**。

Adobe 当前的处境正是这类"夹心层"的典型——Firefly / GenStudio 等 AI 产品体量小但增速快，公司必须持续投资模型与 GPU；Creative Cloud 主业被 Canva / Figma 的 AI 化挤压；同时面对 Sora / Runway 这类前沿视频生成的免费用户分流。CFO 在此时离开（无论是否与战略分歧相关）都会被市场放大解读。

对软件股板块的外溢影响值得关注：Adobe 之外，Salesforce、ServiceNow、Workday 都在同一逻辑下——AI Agent / Copilot 收入若不能在 H2 2026 体现，他们的 PE 倍数都会被重估。下周 Microsoft Build 后续的 Azure 收入分项、7 月 Salesforce 半年报都是关键检验点。

**角度：** 软件股从"AI 受益者"过渡到"AI 受困者"只用了一个季度——核心不是营收，而是 GPU 成本侵蚀传统 SaaS 净利率。

### 🔑 4. 5 月 CPI +4.2%、PPI +6.5% — 通胀粘性 + 伊朗停火，市场押注 Fed 9 月才动手

**[BLS: 5 月 CPI 数据](https://www.bls.gov/news.release/cpi.nr0.htm)** · **[Kraken: CPI/FOMC/SpaceX 时间表](https://blog.kraken.com/economic-brief/june-10-2026)**

6/10 公布的 5 月 CPI 同比 +4.2%、环比 +0.5%，6/11 公布的 PPI 同比 +6.5%、环比 +1.1%——两项数据均显示通胀仍在 4%–6.5% 区间高位粘性。这是 6/16-17 FOMC 会议前最后一组核心通胀数据，市场对"维持 3.50–3.75% 利率"的预期已经接近 100%；真正的悬念在 dot plot 和 Powell 的措辞——是否在 9 月给出明确降息时间窗口。

伊朗停火预期叠加 Brent 原油跌至 $85，给 Fed 提供了"通胀可能见顶"的辩护材料。但服务通胀依然在 5%+，租金分项也未见拐点，Powell 大概率维持"data dependent + 不预先承诺"的措辞。市场已经基本消化"6 月按兵不动 + 9 月降息"的剧本——若 Powell 释放更鸽派信号，纳指可能继续创新高；反之，AI 基建股可能率先承压。

更值得关注的是**美元指数和 10Y 国债收益率的反应不对称**。如果 Fed 释放降息时间表，美元应该走弱、长端利率回落——但 SpaceX、OpenAI、Anthropic 三家巨型 IPO 的资金需求又会推高美元短端流动性需求，形成"短强长弱"的扭曲曲线。这是 2026 H2 最关键的宏观博弈点。

**角度：** Fed 现在面对的不是通胀单线博弈，而是"通胀粘性 vs IPO 资金潮"双线博弈，下周三的 dot plot 会决定 H2 风险资产定价的整个基调。

### 🔑 5. 三大指数本周连续两日反弹 — 反弹的是地缘风险溢价，不是基本面修复

**[The Street: 6/12 收盘](https://www.thestreet.com/stock-market-today/stock-market-today-dow-jones-sp-500-nasdaq-updates-june-12-2026)** · **[CNBC: 6/12 市场](https://www.cnbc.com/2026/06/11/stock-market-today-live-updates.html)**

周五收盘 S&P 500 7,431.46（+0.5%）、Nasdaq 25,888.84（+0.31%）、Dow 51,202.26（+0.7%）。本周两个交易日的反弹合计 S&P +2.25% / Nasdaq +2.85% / Dow +2.56%——但归因看，95% 来自伊朗停火 + 油价回落。AI 基础设施板块（NVDA/AMD/AVGO）今日表现温和，没有显著贡献，传统大市值的能源股反而是拖累项。

板块结构上：科技、工业、原材料领涨；能源、消费必需品、地产承压。这是典型的"风险溢价回落"行情——并非业绩驱动的盈利上修。Magnificent 7 月内蒸发 $2T 市值，本质是被动基金为承接 SPCX/OpenAI/Anthropic IPO 腾出仓位，这部分压力不会因伊朗停火消失。

下周交易日重要节点：6/16 周一 Empire State 制造业、6/17 周二零售销售、6/17–18 FOMC、6/19 周四 BoJ 决议、6/20 周五季度三巫日 (triple witching)。每个节点都有可能成为短线波动放大器，特别是 FOMC 与 BoJ 同周——如果 BoJ 加息 + Fed 维持鸽派 hold，套息交易回流可能引发短期美元抛压。

**角度：** 这一周的反弹是"政治新闻 + 估值修复"的混合体，不是"AI 业绩持续超预期"的延续——下周三 FOMC 之后才是真正的方向选择窗口。
